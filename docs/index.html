Resumen técnico

MercApp es una aplicación web tipo SPA desarrollada con Vue 3 y Vite para el frontend, y Node.js con Express para el backend. El sistema permite visualizar productos, filtrar por categorías y gestionar productos mediante operaciones CRUD.

La base de datos se encuentra alojada en MongoDB Atlas, el backend fue desplegado en Railway, el frontend fue desplegado en Netlify y este micrositio se publica mediante GitHub Pages.

Guía de ejecución local

Backend

cd backend
npm install
npm start

Frontend

cd frontend
npm install
npm run dev

Variables de entorno del backend

PORT=3000
MONGODB_URI=mongodb+srv://usuario:password@cluster.mongodb.net/mercapp
FRONTEND_URL=http://localhost:5173
JWT_SECRET=clave_secreta

Variables de entorno del frontend

VITE_API_URL=http://localhost:3000

Endpoints principales

Método Ruta Descripción Ejemplo
GET / Información general de la API Estado y base de datos activa
GET /health Verifica el estado de salud de la API {"status":"ok"}
GET /productos Lista todos los productos [{"nombre":"Mouse","precio":25}]
POST /productos Crea un producto {"nombre":"Producto prueba","precio":77}
PUT /productos/:id Actualiza un producto por ID {"nombre":"Producto actualizado","precio":88}
DELETE /productos/:id Elimina un producto por ID {"message":"Producto eliminado"}
GET /categorias Lista categorías disponibles [{"id":1,"name":"Tecnologia"}]
