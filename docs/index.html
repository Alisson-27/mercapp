Arquitectura

  1. Vue 3 consume la API REST mediante VITE_API_URL.
  2. Express expone productos, categorias y autenticacion.
  3. Mongoose conecta la API con MongoDB Atlas.
  4. Railway publica el backend y Netlify publica la SPA.

Endpoints principales

  • GET /productos
  • POST /productos
  • PUT /productos/:id
  • DELETE /productos/:id
  • POST /auth/register
  • POST /auth/login
