Resumen técnico

MercApp es una aplicación web tipo SPA desarrollada con Vue 3 y Vite para el frontend, y Node.js con Express para el backend. El sistema permite visualizar productos, filtrar por categorías, agregar productos al carrito y gestionar productos mediante operaciones CRUD.

La base de datos se encuentra alojada en MongoDB Atlas. El backend fue desplegado en Railway, el frontend fue desplegado en Netlify y este micrositio se publica mediante GitHub Pages desde la carpeta /docs del repositorio.

Guía breve de ejecución local

Backend

cd backend
npm install
npm start

El backend se ejecuta localmente en: 

http://localhost:3000

Frontend

cd frontend
npm install
npm run dev

El frontend se ejecuta localmente en: 

http://localhost:5173

Variables de entorno

Backend - Railway o entorno local

PORT=3000
MONGODB_URI=mongodb+srv://usuario:password@cluster.mongodb.net/mercapp
FRONTEND_URL=https://mercappv4.netlify.app
CORS_ORIGIN=https://mercappv4.netlify.app
JWT_SECRET=clave_secreta

Frontend - Netlify

VITE_API_URL=https://mercapp-production-ba3d.up.railway.app

Nota: Las credenciales reales no se versionan en GitHub. El repositorio incluye archivos .env.example sin datos sensibles.

Endpoints principales

Método Ruta Descripción Ejemplo de request/response
GET / Información general de la API {"name":"MercApp API","status":"ok"}
GET /health Verifica el estado de salud de la API {"status":"ok","database":"mongodb"}
GET /productos Lista todos los productos [{"nombre":"Mouse","precio":25}]
POST /productos Crea un nuevo producto {"nombre":"Producto prueba","precio":77}
PUT /productos/:id Actualiza un producto por ID {"nombre":"Producto actualizado","precio":88}
DELETE /productos/:id Elimina un producto por ID {"message":"Producto eliminado"}
GET /categorias Lista las categorías disponibles [{"id":1,"name":"Tecnologia"}]
